EnvoiMoinsCher API - la classe pour la gestion des pays.

La classe peut être utilisée pour le chargement d'un pays ou de toute la liste des pays avec les codes ISO, utilisés par EnvoiMoinsCher.com pour effectuer des devis.

"VI" => "V", "GR" => "G") (line 19)

Le tableau protégé avec les relations entre les pays et leurs codes ISO.


Il est utilisé pour établir un code ISO de certaines destinations. Par exemple, en récupérant les informations sur l'Espagne (code ISO unique "ES"), on recevra aussi les informations sur les Canaries (code ISO unique "E1", mais appartenant à l'Espagne). Cette dépendance est utilisée par la méthode getCountry($code))

  • access: protected

Le tableau avec tous les pays. La clé de chaque entrée est le code ISO unique du pays.
Par exemple pour l'entrée avec l'Argentine va se traduire ainsi dans le langage PHP :
