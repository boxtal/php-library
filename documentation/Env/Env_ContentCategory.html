array $categories = array() (line 23)

La variable publique contenant la liste des catégories. Les identifiants des catégories servent comme clés du tableau.

Par exemple, on récupère une catégorie avec les valeurs suivantes :
- identifiant : 1111
- code : A
- libellé : First category A
Le tableau PHP correspondant à cette catégorie sera le suivant :
$categories = array(1111 => array("label" => "First category A", "code" => "A"))

  • access: public
array $contents = array() (line 31)

La variable avec les contenus (sous-catégories). Chaque contenu est attaché à une catégorie par l'identifiant de cette catégorie.

Par exemple on récupère la catégorie dont l'identifient est égal à 1111. Les contenus de cette catégorie vont être représentés de la manière suivante dans un tableau PHP:
$categories[1111] = array(0 => array("code" => 11112, "label" => "code 11112"), 1 => array("code" => 11113, "label" => "code 11113"))
